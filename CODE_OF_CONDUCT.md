# Code of Conduct

We pledge to make participation a harassment-free experience for everyone.
Be respectful, assume good faith, give constructive feedback, and focus on what
is best for the community. Harassment, trolling, and publishing others' private
information are not tolerated.

Report issues to the maintainer via https://github.com/aj-2-c-2-a.
Adapted from the Contributor Covenant v2.1.
